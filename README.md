## About Project

This project provides starting point for fresh laravel instalation. It has pre-configured Users/Roles and Permissions CRUD. 
Built on AdminLTE.

## Installation Commands

php composer.phar update
php composer.phar install
php artisan key:generate
php artisan migrate:refresh --seed
